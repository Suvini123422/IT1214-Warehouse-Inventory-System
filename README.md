Project description............................................
The Warehouse Inventory System is a simple console-based Java application that helps manage products stored in a warehouse. It allows users to:

View all available products
Add new products
Update product quantities

The system uses arrays to store:

Product names
Quantities
Prices

How to run......................................................
javac WarehouseInventorySystem.java
java WarehouseInventorySystem

Sample output...................................................
1. View All
2. Add Product
3. Update Quantity
0. Exit
